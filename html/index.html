Novamedia innovision

Zajmuję się pełnym procesem produkcyjnym oprogramowania: od zapoznawania się z SIWZ i określania wymagań, przez pisanie i kompilację kodu, po dystrybucję i diagnostykę. Aplikacje które pisałem wykorzystywały technologie komunikacji takie jak strumienie standardowego wyjścia/wejścia, nazwanych, socketów, ramek TCP, komunikacji zdarzeniowej MQTT, wbudowanego C Python API, modułów boost::python. Konsekwentnie aplikacje musiały być wielowątkowe. Używałem standardowych stylów formatowania, np oferowanych przez clang-format, czy PEP8. Implementowałem obsługę API partnerów biznesowych. Przygotowywałem aplikacje prezentacji wg. projektu graficznego, wykorzystujące openGL (metodą glVertex*), SDL2 z pomocą silnika zarządzającego atlasami tekstur, bibliotekę realizującą elementarne struktury graficzne (obrazy, etykiety, tilemapy), HTML i javascript.
Sukcesy: Miałem możliwość modernizacji wielu sfer działania firmy}, między innymi implementując i popularyzując ujednolicony sposób budowania, realizacji obsługi standardowego protokołu, zwiększając produktywność kolegów i redukując liczbę problemów związanych z mnogością implementacji. Realizowałem założenia projektów aplikacjami wbudowanymi, które są używane do dziś i są gotowe na wyzwania przyszłości.
